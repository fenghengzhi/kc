名称及摘要

中

一种大数据使用权交易系统与方法
本发明公开了一种大数据使用权交易系统与 方法，该系统包括产业链生态大数据模型模块、大数据生态链应用模块、大数据生态链接口模块、 交易撮合模块、大数据生态链价值模型模块以及大数据交易平台模块。 该方法包括选取目标产业链完成大数据交易系统的构建等六步骤。 该大数据使用权交易方法按照产业生态体系大数据模型来直接实现大数据应用， 实现在大数据交易过程中的使用权交易，在交易中直接完成大数据价值应用。

英

Big data use right transaction system and method
The invention discloses a big data use right transaction system and a method. The system comprises an industry chain ecological big data model module, a big data ecological chain port module, a transaction matching module, a big data ecological chain value model module and a big data transaction platform module. The method comprises six steps of selecting a target industry chain to finish construction of a big data transaction system, and the like. By use of the big data use right transaction system method, big data application is directly achieved according to a big data model of an industrial ecological system; use right transaction in a big data transaction process is achieved; and big data value application in the transaction is directly finished.
